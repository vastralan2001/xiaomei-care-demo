## 使用说明

### H5 移动端（推荐手机体验）

1. 手机浏览器打开链接，或微信扫码
2. **顶部 Tab 切换 4 个场景**：🍔不健康饮食 / 🌧️天气关怀 / 🌙深夜加班 / 💆情绪关怀
3. **A/B 分组切换**：点击「对照组 G1」或「实验组 G3」，观察两组体验差异
4. **底部快捷提问**：点击不同 Query 触发对话

### 对话流程（实验组 G3）

```
小美打招呼 → 用户提问 → 小美正常回复 → 推荐 3 家普通店铺
                                          ↓
                                ─── 小美关怀 ───
                                          ↓
                              关怀话术（自然衔接）
                                          ↓
                              推荐 1 家关怀店铺（带标签 + 菜品）
                                          ↓
                              红包券（可点击「立即领取」）
```

### 对照组 G1 vs 实验组 G3

- **G1 对照组**：正常推 3 家店 → 结束，无关怀内容
- **G3 实验组**：正常推 3 家店 → 追加关怀话术 + 1 家关怀店 + 红包券

### 桌面完整版

左侧场景列表 + 中间手机模拟 + 右侧策略面板（触发条件、搜索 Query、Prompt 模板、券配置、AB 实验设计）

核心链路：`Query → CentralAgentRunner → HttpToolkit → callTuanSearchApi(主搜3店) + searchHealthPoi(并行关怀店) → PoiSummary + health_context 注入 → LLM 生成话术 + 券 mock`
